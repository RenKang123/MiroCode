2020.06.20 关于雾化头的控制
缺水后雾化头会自动停止工作，并输出低电平；
单片机不需要控制关闭，按照正常模式工作即可
电源板有晶振，所以定时放在电源板
2020.08.03 关于WIFI控制开机和关机指令
电源板发给显示板的指令有两个标志位来控制开机或者关机 位1代表需要操作
若WIFI给出关机指令，需要查询电源板状态，若电源板处于关机状态，则不需要发送指令给显示板
若电源板处于开机状态，则需要发送指令，直到电源板是关机状态则不发送指令
开机也是类似的操作
2020.08.06 HARU 需要掉电记忆
关于HARU 重新定义 
1. 湿度显示以固定值进行 
    1) 干燥 : 0~40% 
    2) 舒适 : 40%~60%
    3) 潮湿 : 60%~
2. Auto Mode 时 加湿强度工作 
    1) 加湿 3档 : 0~ HARU3 
    2) 加湿2档 : HARU3 ~ HARU 2
    3) 加湿1档 : HARU2~ HARU 1
    4) 加湿强度 OFF : HARU1 ~ 100
